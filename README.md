# radicaster

radicasterは、AWS上で動作するradikoを録音・Podcast化するアプリケーションです。

## 特徴

- マネージドサービスを活用したアーキテクチャにより低コストで運用可能
- AWS CDKによる簡単なデプロイ
- 分割された番組の結合に対応

## 主な使用サービス

- Lambda
- EventBridge
- S3
- CloudFront
